# 🏠 Homelab Ansible

Infrastructure-as-Code (IaC) repository for automating and managing my homelab using **Ansible**.

This project serves as a centralized source of truth for provisioning, configuring, and maintaining services across my homelab environment, including containers, networking, monitoring, and system configuration.

## 🚀 Overview

This repository contains Ansible playbooks, roles, and inventories used to:

- Automate provisioning of systems (VMs, LXCs, or bare metal)
- Configure services and applications
- Maintain consistency across environments
- Reduce manual setup and configuration drift

The goal is to make the homelab **reproducible, scalable, and easy to manage**.

## 🧰 Technologies Used

- **Ansible** – Configuration management & automation  
- **Docker** – Containerized services  
- **Linux** – Host systems (Ubuntu/Debian-based)  
- **Proxmox / LXC (optional)** – Virtualization layer  
- **YAML / Jinja2** – Configuration templates
